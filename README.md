# Traffic Light Java Simulator

This project is a Java Swing GUI application that simulates a traffic light system using event-driven programming.

## Features
- Simulates real-world traffic light transitions (Red, Yellow, Green)
- Graphical user interface built with Java Swing
- Event-driven programming logic
- Modular object-oriented design

## Technologies Used
- Java
- Java Swing
- Object-Oriented Programming (OOP)

## Purpose
This project was developed as part of coursework at the University of Maryland Global Campus to demonstrate GUI programming and state-based logic control.
